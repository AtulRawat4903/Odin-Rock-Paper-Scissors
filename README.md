# ✊ Rock Paper Scissors

A browser-based Rock Paper Scissors game built with HTML, CSS, and JavaScript. Play against the computer by clicking buttons, track the score in real time, and keep playing until one player reaches 5 points.

## 🚀 Features

* Interactive button-based gameplay
* Random computer choices
* Live score tracking
* Winner announcement when a player reaches 5 points
* Play Again button to restart the game
* Buttons disabled when the game ends
* Responsive, modern dark-themed UI

## 🖥️ Demo

Open `index.html` in any modern web browser to play.

## 🎮 How to Play

1. Click **Rock**, **Paper**, or **Scissors**.
2. The computer randomly selects its move.
3. The winner of the round is displayed.
4. Scores are updated after each round.
5. The first player to reach **5 points** wins the game.
6. Click **Play Again** to start a new match.

## 📌 Game Rules

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock
