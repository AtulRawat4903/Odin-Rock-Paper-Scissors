✊ Rock Paper Scissors Game
A simple browser-based Rock Paper Scissors game built using HTML, CSS (optional), and JavaScript. The player competes against the computer for 5 rounds, and the final score is displayed in the console.


🚀 How to Run
* Download or clone the project
Open the folder
Open index.html in any modern browser
Play the game via the prompt dialog
Check results in the browser console (F12 → Console)


🎮 How It Works
User inputs: Rock / Paper / Scissors
Computer choice is generated randomly using Math.random()
The game runs for 5 rounds
Scores are tracked for both player and computer
Final winner is displayed in the console


🧠 Technologies Used
HTML5
JavaScript (Vanilla)


📌 Game Rules
Rock beats Scissors
Scissors beats Paper
Paper beats Rock


📊 Example Output
You Win! rock beats scissors
You Lose! paper beats rock
It's a tie! Both chose scissors

Final Score:
Human: 3
Computer: 2

🏆 You win the game!
